# Tweets-Collection-and-Indexing
UB CSE 535 Information Retrieval

Collected tweets on different topics and different languagues using twitter's REST API and Streaming API

Also converted the raw JSON files obtained into required format that can be indexed. This indexind involves the following steps:

Extracting Hash-Tags, Mentions, Emoticons and Coordinates(Location).
Text santizing
Classification based on the Language of the Tweet

Input: JSON file in RAW format containing the Tweets
Output: JSON file with all the pre-processing done and ready to be indexed
